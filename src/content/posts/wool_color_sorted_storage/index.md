# 基本信息
全颜色羊毛仓库是一个大宗仓库在海淀工业区，而机器本体不在的一个配套，其拥有全颜色的羊毛的生产/储存一体模式为日啊服务器最速、最大、~~最实用~~、最多 的羊毛机

可产出所有颜色的羊毛并储存，其耗材为`盒子`与`铁锭`，抗卸载，需要真人挂机

# 仓库位置
由于机器位置并不在海淀的核心区，但是仓储部分在，所以直接把仓库写在前面，因为用户根本不需要跑到机器本体去

机器精准位置: `x-6256 z-9192`
